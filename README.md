# Gym Tracker

App web simple para llevar control de ejercicios de gimnasio.

## Uso

1. Abre `index.html` en tu navegador.
2. Registra fecha, ejercicio, series, reps y peso.
3. Revisa historial, resumen y récords por ejercicio.

## Funciones

- Guardado local automático (`localStorage`).
- Filtro por nombre de ejercicio.
- Cálculo de volumen total.
- Eliminación individual o borrado total.
